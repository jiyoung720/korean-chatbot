# Evaluation Dataset v1

RAG 코퍼스(한국어 위키백과 1,000문서)를 기반으로 답할 수 있는 질문과
모범 답안 10개입니다. 엘라 모델(직접 구현), Gemma 4 e2b(공개 가중치),
Gemini 2.5 Flash(프론티어 API) 세 모델을 동일한 RAG 파이프라인
(`rag/shared/`)으로 평가하기 위한 공통 기준입니다.

카테고리(인물 3, 국가/지역 2, 개념 3, 과학 1, 프로젝트 연계 1)와 질문
유형(정의형, 관계형, 위치형, 특징형, 업적형)을 고르게 섞었습니다.

## Q1 (인물 / 정의형)

질문:
지미 카터는 누구인가?

모범답안:
미국의 제39대 대통령(1977년~1981년)이며, 민주당 출신이다.

## Q2 (인물 / 특징형)

질문:
백남준은 어떤 예술 분야로 유명한가?

모범답안:
한국 태생의 세계적인 비디오 아트 예술가이자 작곡가, 전위 예술가로
유명하다.

## Q3 (인물 / 관계형)

질문:
데니스 리치는 어떤 프로그래밍 언어 개발에 기여했는가?

모범답안:
C 언어를 만들었으며, 유닉스 운영체제 개발에도 기여한 컴퓨터과학자이다.

## Q4 (국가/지역 / 위치형)

질문:
우크라이나는 어디에 위치한 국가인가?

모범답안:
동유럽에 위치한 국가로, 남쪽은 흑해와 아조프해, 동쪽은 러시아, 북쪽은
벨라루스와 접한다.

## Q5 (국가/지역 / 위치형)

질문:
체첸 공화국은 어느 지역에 위치하는가?

모범답안:
러시아를 이루는 공화국 중 하나로, 북캅카스 지역에 위치한다.

## Q6 (개념 / 정의형)

질문:
수학은 무엇을 연구하는 학문인가?

모범답안:
수, 양, 구조, 공간, 변화 등의 개념을 다루는 학문이다.

## Q7 (개념 / 정의형)

질문:
화학은 무엇을 연구하는 학문인가?

모범답안:
물질의 성질, 조성, 구조, 변화 및 그에 수반하는 에너지의 변화를
연구하는 자연과학의 한 분야이다.

## Q8 (개념 / 정의형, 다소 어려움)

질문:
초월수란 무엇인가?

모범답안:
유리수 계수를 가지는 0이 아닌 유한 차수 다항 방정식의 해가 될 수 없는
수, 즉 대수학적이지 않은 수를 의미한다.

## Q9 (과학 / 정의형)

질문:
맥스웰 방정식은 무엇인가?

모범답안:
전기와 자기의 발생, 전기장과 자기장, 전하 밀도와 전류 밀도의 형성을
나타내는 4개의 편미분 방정식이다.

## Q10 (프로젝트 연계 / 업적형)

질문:
세종대왕은 어떤 업적으로 유명한가?

모범답안:
1443년 한글(훈민정음)을 창제하기 시작하여 1446년 반포했다.

(이 프로젝트에서 Vanilla RAG와 LangChain RAG 실험 모두 "세종대왕은
누구야?"를 반복 테스트 질문으로 사용했으므로, 프로젝트 히스토리와
직접 연결되는 항목으로 포함)

## 평가 기준 (초안)

각 응답을 다음 기준으로 0~5점씩 채점합니다(총 15점).

- **정확성**: 모범답안과 사실 내용이 일치하는가
- **질문 관련성**: 질문에서 요구한 정보(정의/관계/위치/특징/업적)에
  정확히 답하는가, 아니면 무관한 내용이 섞여 있는가
- **자연스러움**: 문법적으로 통하는 완성된 문장인가

채점은 사람이 직접 하거나, Gemini 2.5 Flash를 Judge 모델로 사용해
"해당 분야의 전문가" 페르소나를 주고 평가시키는 방식도 고려합니다
(`docs/mentoring_notes.md`의 "AI를 judge로 쓸 때" 참고).

## 평가 방법

각 질문에 대해 세 모델(엘라/Gemma 4 e2b/Gemini 2.5 Flash)의 RAG 응답을
같은 `rag/shared/` 파이프라인(검색)으로 수집하고, 다음 형식의 비교표로
정리합니다.

| ... | | | |

## 향후 확장 계획

- v1: 10문항 (현재) — 모두 Single-hop(검색된 문서 하나로 답이 나오는 유형)
- v2: 30문항으로 확장 — 더 다양한 유형/난이도 추가. Multi-hop 질문
  도입 검토(예: "지미 카터와 우크라이나는 어떤 관련이 있는가?"처럼
  문서 여러 개를 조합해야 답이 나오는 유형). Single-hop → Multi-hop
  확장은 추후 Graph RAG/Agent 단계와도 연결되는 지점
- v3: 50문항, RAG 없이 모델만 비교하는 항목도 추가 (`prompt | model |
  parser` 형태, `docs/mentoring_notes.md` 참고)
